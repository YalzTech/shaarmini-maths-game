# Maths Millionaire Quiz Game

A simple GitHub Pages quiz game for Malaysian Primary Year 1 and Year 2 Mathematics level.

## Included topics

- Numbers and counting
- Addition
- Subtraction
- Multiplication basics
- Division/sharing basics
- Money
- Time
- Shapes
- Measurement

## How to edit questions

Open `script.js` and edit the `questions` array.

Correct answer guide:
- `correct: 0` = A
- `correct: 1` = B
- `correct: 2` = C
- `correct: 3` = D

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, and `script.js`.
3. Go to Settings.
4. Go to Pages.
5. Under Branch, choose `main`.
6. Save.
